# 📊 Web Traffic Analytics Project

## 📌 Overview
This project focuses on analyzing website user data to understand engagement, conversion rate, and drop-off points.

## 🎯 Objective
- Analyze user journey from view → add to cart → purchase  
- Identify conversion gaps  
- Improve user engagement  

## 🛠 Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)

## 📊 Key Metrics
- Total Users
- Total Revenue
- Conversion Rate
- Bounce Rate

## 📉 Funnel Analysis
The funnel shows user drop-off at each stage:
View → Add to Cart → Purchase

## 📸 Dashboard Preview
![Dashboard](images/dashboard.png)

## 🔍 Key Insights
- High number of users drop before purchase  
- Conversion rate is low compared to views  
- Some categories perform better than others  

## 🚀 Conclusion
This project highlights how data can be used to identify problem areas and improve decision-making.
